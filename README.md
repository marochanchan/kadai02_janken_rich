## ⑤工夫した点・こだわった点
- タイトル画面→ゲーム画面といったようにシーンが遷移するようにしました。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 構想をAIに相談→枠組みを提案→自力で実装という流れにしたかったのですが、そもそもの構造の組み立てや、シーン・テキストを自然に表示するのが難しく、AIの力を結構借りないと実装できませんでした。
- シーンの切り替えについては、Aの画像をBに変更するという単純な指示だけではなく、一度フェードアウトさせて消してから別の画像を自然にフェードインさせるなど、自分の想像以上に細かく分解して考えなければいけないところに難しさを感じました。
- BGMや効果音もあったらよりゲームっぽくなるのかなと思いました。
